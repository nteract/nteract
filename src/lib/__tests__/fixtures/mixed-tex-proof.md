# A tidy little proof

Suppose sqrt(2) is rational, so $\sqrt{2}=p/q$ for integers in lowest terms.

\begin{align}
\sqrt{2} &= \frac{p}{q} \\
2q^2 &= p^2
\end{align}

The equation shows p is even, so write p = 2k.

\begin{equation}
q^2 = 2k^2
\end{equation}

Now q is even too, contradicting that p/q was in lowest terms.
