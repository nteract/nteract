Site-like host chrome uses the Unified Site Directions language for cloud
surfaces that sit around notebooks: the Mono Ledger header, mono eyebrow rows,
tight headlines, square detail language, and Ink-forward app shells.

This language does not apply to the notebook app view. Cells, toolbar, outputs,
rails, and the shared notebook shell keep their existing notebook-specific
surface language. Cloud sign-in, home, empty, blocked, not-found, and
session-expired screens adopt this pattern as needed.
