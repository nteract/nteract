## Boundary

- The shell, rail, command toolbar, comments panel, package panel, workstation
  panel, and `NotebookView` come from shared production components.
- Elements owns only the fixture scenario, comments projection, focus state,
  and inert host callbacks.
- Desktop and cloud should continue to pass host facts into the same shared
  surfaces instead of forking shell layout logic.
