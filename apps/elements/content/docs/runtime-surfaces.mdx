## Host notes

The current previews keep side effects inert. Components that need host access,
such as the pool settings action, receive adapter callbacks instead of importing
notebook shell state into the docs app. The boundary table names the split
between deterministic preview data, host actions, runtime side effects, and live
package/trust documents.
