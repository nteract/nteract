Cloud viewer unhappy paths use one visual family: centered copy, a quiet icon
badge, one sentence of detail, and at most one action. Severity changes the icon
and subtle badge or border tone, not the whole surface. Severity colors reuse
the runtime state language: kernel-starting blue for info, executing amber for
attention, destructive red for broken, live-presence green for success. These
fixtures keep the states calm across empty, blocked, stale, loading, and
signed-out paths. Titles are short labels without trailing periods; details are
single sentences.
